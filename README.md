# blockscan-label-crawler

Crawls all Blockscan.com explorer address labels and saves them to a JSON file, including:

- arbiscan.io
- beaconscan.com
- bobascan.com
- bttscan.com
- etherscan.io
- celoscan.xyz
- clvscan.com
- ftmscan.com
- gnosisscan.io
- polygonscan.com
- moonriver.moonscan.io
- moonscan.io
- optimistic.etherscan.io
- bscscan.com
- snowtrace.com

## Install

```
npm i
mkdir -p data
```

## Start

```
npm run start
```
